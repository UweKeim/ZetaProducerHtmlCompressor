Praktikanten/Werkstudenten (w/m)
für die .NET-Software-Entwicklung

Ihre Aufgaben

  • Sie arbeiten aktiv mit an Kunden- und Forschungsprojekten Sie erarbeiten Wissens-Know-How in neuen Technologien
  • Sie betreuen und erweitern bestehende Anwendungen

Es erwartet Sie ein modernes und freundliches Umfeld mit einem dynamischen und engagierten Team von Entwicklern und Projektmanagern, die Spaß an ihrer Arbeit haben.

Ihr Profil

  • Sie absolvieren ein Studium der Informatik Sie besitzen Kenntnisse in einer oder mehreren der folgenden Technologien: SharePoint, Navision, C#, Windows Forms, XAML, ASP.NET, SQL, AJAX, Silverlight, .NET Framework 3.0 (Workflow Foundations, Pre-sentation Foundations)
  • Sie haben reichlich Neugier, Kreativität und soziale Kompetenz
  • Sie haben 15-20 Stunden die Woche Zeit
