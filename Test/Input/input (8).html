Bewerbungstest Fachinformatiker

Bearbeiten Neu Erweitert

Aufgabe:

Schreiben Sie in C# eine Windows Forms-Anwendung, die mehrere Strings abfrägt und diese in einem XML-Dokument abspeichert.

Hinweise:

  • Diese Aufgabe dient nur der Ermittlung der Programmier-Kenntnisse eines Bewerbers. Die Ergebnisse werden nicht kommerziell verwendet.
  • GUI-Gestaltung und XML-Format können/müssen selbst definiert werden.
  • Es werden keine weiteren Informationen zur Spezifikation bereitgestellt.
  • Das Programm senden Sie uns bitte im Quelltext (Zip-Datei) gemeinsam mit Ihrer Bewerbung per E-Mail.

Bitte beachten Sie unsere Bewerbungsinformationen.
